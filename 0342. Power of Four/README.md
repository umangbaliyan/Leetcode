# Power of Four

## Problem Statement

Given an integer `n`, return `true` if it is a power of four. Otherwise, return `false`.

An integer `n` is a power of four if there exists an integer `x` such that:

`n = 4^x`

## Examples

### Example 1

**Input:**
```text
n = 16
```

**Output:**
```text
true
```

### Example 2

**Input:**
```text
n = 5
```

**Output:**
```text
false
```

### Example 3

**Input:**
```text
n = 1
```

**Output:**
```text
true
```

## Constraints

```text
-2^31 <= n <= 2^31 - 1
```
