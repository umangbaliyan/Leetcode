# Add Digits

## Problem Statement

Given an integer `num`, repeatedly add all its digits until the result has only one digit, and return it.

## Example 1

**Input:**
```text
num = 38
````

**Output:**

```text
2
```

**Explanation:**

```text
38 --> 3 + 8 --> 11
11 --> 1 + 1 --> 2
```

Since `2` has only one digit, return it.

## Example 2

**Input:**

```text
num = 0
```

**Output:**

```text
0
```

## Constraints

* `0 <= num <= 2^31 - 1`
